# 📡 KKTCELL Homepage Clone (React + Tailwind)

This project is a **frontend clone of the Kuzey Kıbrıs Turkcell (KKTCELL) homepage**, built using **React and Tailwind CSS**.

It is created for **educational purposes only**. All design rights belong to the original company.

---

## 🌐 Live Demo

👉 https://turkcell-website-clone.vercel.app/

## 🚀 Features

- ⚛️ React component-based architecture
- 🎨 Tailwind CSS modern styling
- 📱 Fully responsive design
- 🧩 Reusable UI components
- ✨ Hover animations and interactive UI effects
- 📌 Fixed sidebar popup elements
- ⚡ Fast and lightweight structure
